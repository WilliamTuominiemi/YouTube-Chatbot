# YouTube Chatbot

## About

A chatbot for your YouTube livestream

## Features

-   Subcount
-   Commands

    | Command  | Response                       |
    | -------- | ------------------------------ |
    | '/help'  | `Sends list of commands`       |
    | '/stats' | `Sends channel stats`          |
    | '/dc'    | `Sends link to discord server` |
